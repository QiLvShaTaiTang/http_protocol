范围请求

以前没有高带宽的互联网，下载一个尺寸稍微大的文件都会很吃力，如果遇到网络终端则要重头开始
范围请求：可恢复机制，对下载的内容范围可以设置，从中断的地方重新开始
内容协商

同一个Web网站会存在多个相同页面，例如：不同的语言的页面
内容协商机制：服务器端和客户端对响应的资源内容进行交涉返回最适合的资源

Accept
Accept - Charset

三种协商类型：
服务器驱动协商（server自动处理）
客户端驱动协商（用户选择）
透明协商（两种驱动结合）
